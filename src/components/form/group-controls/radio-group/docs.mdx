---
componentName: RadioGroup
---

Composes multiple `<RadioField>` components into a radio group. Can be used to select a single choice from a list of options. Styled to match the [HashiCorp Design System spec](https://design-system-components-hashicorp.vercel.app/components/form/radio).

## `isRequired` and `isOptional`

### With `legend` and `isRequired`

<RadioGroup
	name="group1"
	isRequired
	legend="Legend of the group"
	options={[
		{
			label: 'Label of control #1',
			value: '1',
		},
		{
			label: 'Label of control #2',
			value: '2',
		},
		{
			label: 'Label of control #3',
			value: '3',
		},
	]}
/>

### With `legend` and `isOptional`

<RadioGroup
	name="group2"
	isOptional
