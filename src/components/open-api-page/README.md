# OpenApiPage

The `<OpenApiPage />` component, and related server utilities, are used to render documentation pages for a provided [Swagger OpenAPI 2.0 specification](https://swagger.io/specification/v2/).

## Installation

```sh
npm install @hashicorp/react-open-api-page
```

## Usage

See usage examples in [docs.mdx](docs.mdx), or in [Swingset](http://react-components.vercel.app/components/openapipage).
