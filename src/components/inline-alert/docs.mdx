The `InlineAlert` component is used in long-form MDX content to highlight various message types. Default titles and icons are provided for each 'type', however these can be overridden via the `title` prop.

## Props

| Prop name  | Description                                                            | Required? | Default value      | Type                             |
| ---------- | ---------------------------------------------------------------------- | --------- | ------------------ | -------------------------------- |
| `children` | The alert body text to render.                                         | Yes       | undefined          | string                           |
| `type`     | The 'type' which has associated styles and default data (icon, title). | No        | tip                | tip, highlight, warning, or note |
| `title`    | An option to override the default title.                               | No        | The type's default | string                           |

## Alert types

### Tip

<InlineAlert>
	To provide general information to the user regarding the current context or
	relevant actions.
</InlineAlert>

### Highlight

<InlineAlert type="highlight">
	To provide general or promotional information to the user prominently.
</InlineAlert>

### Note

<InlineAlert type="note">
	To help users avoid an issue. Provide guidance and actions if possible.
</InlineAlert>

### Warning

<InlineAlert type="warning">
	To indicate critical issues that need immediate action or help users
	understand something critical.
</InlineAlert>
