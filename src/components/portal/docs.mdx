`<Portal />` renders its children using [`ReactDOM.createPortal`](https://reactjs.org/docs/portals.html).

<LiveComponent>{`
<Portal>
  <div
    style={{
      position: "absolute",
      top: "300px",
      left: "calc(50% - 100px)",
      border: "5px solid red",
      borderRadius: "5px",
      padding: "5px",
      width: "200px",
      background: "white",
    }}
  >
    I'm rendered outside of the DOM tree of my parent, allowing for use of
    absolute positioning, for example.
  </div>
</Portal>
`}</LiveComponent>
