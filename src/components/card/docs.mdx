## Props

| Prop name   | Description                                                                | Required? | Default value | Type                          |
| ----------- | -------------------------------------------------------------------------- | --------- | ------------- | ----------------------------- |
| `children`  | The element(s) to render within the `Card`.                                | Yes       | N/A           | ReactNode                     |
| `className` | A string of one or more class names. Applied last to the `Card` container. | No        | undefined     | string                        |
| `elevation` | The elevation of the `hds-surface` class to apply to the `Card` container. | No        | "mid"         | One of: "base", "mid", "high" |
