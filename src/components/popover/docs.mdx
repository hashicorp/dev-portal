This is a rough cut at a Popover component. This implementation is largely duplicative of the tooltip partial in [`@hashicorp/react-logo-grid`](https://github.com/hashicorp/react-components/tree/main/packages/logo-grid/partials/popover-tooltip).

## Examples

`theme="light"`

<>
  {(() => {
    const [showDialog, setShowDialog] = React.useState(false)
    const triggerRef = React.useRef(null)
    return (
      <>
        <button ref={triggerRef} onClick={() => setShowDialog(!showDialog)}>
          Show an example popover
        </button>
        <Popover
          theme="light"
          triggerRef={triggerRef}
          shown={showDialog}
          setIsShown={setShowDialog}
        >
          Hello popover content.
        </Popover>
      </>
    )
  })()}
</>
