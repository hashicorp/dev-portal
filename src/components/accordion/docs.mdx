componentName: 'Accordion'
---

The `<Accordion />` component renders an array as items as a group of disclosures. It accepts an array of `{ title, content }` items to render.

The `<Accordion />` component also accepts an optional `activatorHeadingLevel` prop. This prop is intended to allow the consumer to establish appropriate semantics for the `Accordion` in the context of the page. See the ARIA Authoring Practices Guideline page on [the Accordion pattern](https://www.w3.org/WAI/ARIA/apg/patterns/accordion/) for reference.

## Playground

<LiveComponent>{`
<Accordion
  activatorHeadingLevel="h3"
	items={['Item One', 'Another Item Two', 'Third Item'].map((text) => {
		return {
			title: text,
			content: (
				<div
					style={{
						display: 'flex',
						alignItems: 'center',
						justifyContent: 'center',
						minHeight: '400px',
						background: 'lightgray'
					}}
				>
					Here's some content for {text}
				</div>
			),
		}
	})}
/>
`}</LiveComponent>

## Examples

### Single Item

<Accordion
	activatorHeadingLevel="h4"
	items={['foo'].map((text) => {
		return {
			title: `${text[0].toUpperCase()}${text.slice(1)} accordion item`,
			content: (
				<div
					style={{
						display: 'flex',
						alignItems: 'center',
						justifyContent: 'center',
						minHeight: '400px',
						background: 'lightgray',
					}}
				>
					Hello {text} disclosure content.
				</div>
			),
		}
	})}
/>

### Nested Items

<Accordion
	activatorHeadingLevel="h4"
	items={['there', 'are', 'nested', 'items', 'within'].map((text) => {
		function getPlaceholderItems(textArray) {
			return textArray.map((text) => {
				return {
					title: `${text[0].toUpperCase()}${text.slice(1)} accordion item`,
					content: (
						<div
							style={{
								display: 'flex',
								alignItems: 'center',
								justifyContent: 'center',
								minHeight: '400px',
								background: 'lightgray',
							}}
						>
							Hello {text} disclosure content.
						</div>
					),
				}
			})
		}
		return {
			title: `${text[0].toUpperCase()}${text.slice(1)}`,
			content: (
				<div
					style={{
						display: 'flex',
						flexDirection: 'column',
						gap: '24px',
					}}
				>
					<Accordion
						activatorHeadingLevel="h5"
						items={getPlaceholderItems(['hey'])}
					/>
					<Accordion
						activatorHeadingLevel="h5"
						items={getPlaceholderItems(['these', 'here', 'are'])}
					/>
					<Accordion
						activatorHeadingLevel="h5"
						items={getPlaceholderItems(['nested', 'items'])}
					/>
				</div>
			),
		}
	})}
/>
