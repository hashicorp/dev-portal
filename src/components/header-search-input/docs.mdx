The `HeaderSearchInput` is intended to be a version of our existing [`@hashicorp/react-search`](https://react-components.vercel.app/components/search) component that uses alternate styling for both the input, and for hits.

## Look and feel

The ability to re-skin our existing Search input still needs to be implemented. Differences in HTML structure required for efficient style application may mean we need be to rebuild our search input from the ground up, using what we have in [`@hashicorp/react-search`](https://react-components.vercel.app/components/search) as a starting point.
