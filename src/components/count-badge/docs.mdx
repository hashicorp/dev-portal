## Props

| Prop name | Description                                          | Required? | Default value | Type                                   |
| --------- | ---------------------------------------------------- | --------- | ------------- | -------------------------------------- |
| `color`   | Similar to `Badge`'s `color` prop, but less options. | No        | "neutral"     | One of: "neutral", "neutral-dark-mode" |
| `size`    | Same as `Badge`'s `size` prop.                       | No        | "medium"      | See `BadgeProps`                       |
| `text`    | Similar to `Badge`'s `text` prop, but is required.   | Yes       | N/A           | See `BadgeProps`                       |
| `type`    | Same as `Badge`'s `type` prop.                       | No        | "filled"      | See `BadgeProps`                       |
