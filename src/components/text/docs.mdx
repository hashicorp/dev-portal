The `Text` component should be used for easily styling `li`, `p`, and `span` elements to conform to the design system.

## Props

In addition to the props described in the table below, this component also accepts any other valid props for its `asElement`. A caveat:

- The given `className` prop is applied last to the list of class names set on the rendered element. This means that some CSS properties applied with the following props can be overwritten if needed (not recommended).

| Prop name   | Description                                             | Required? | Default value | Type                                            |
| ----------- | ------------------------------------------------------- | --------- | ------------- | ----------------------------------------------- |
| `asElement` | The name of the element tag to render.                  | No        | "p"           | One of: "li", "p", "span"                       |
| `size`      | The number of the `hds-typography-body` style to apply. | No        | 300           | One of: 100, 200, 300                           |
| `weight`    | The name of the `hds-font-weight` style to apply.       | No        | "regular"     | One of: "regular", "medium", "semibold", "bold" |
